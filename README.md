# uestc_slide_touying
### 介绍
这是一个电子科技大学的 Touying 模版，适用于快速构建组会汇报 slide。
### 文件说明
```text
uestc_slide_touying/
├── main.typ               # 主入口文件
├── ref.bib                # Bib 文献数据库
├── main.pdf               # 输出 PDF
│
├── themes/                # 自定义主题组件
│   └── uestc.typ
│
├── pictures/              # 所有插图、logo 等静态资源
│   ├── uestc-logo.svg
│   ├── uestc-name.svg
│   └── ...                # 其它图像
│
├── docs/
│   └── main.pdf           # 在网页端预览输出的 PDF
│
├── folder-alias.json      # alias插件用于注释文件（未上传）
└── README.md              # 项目说明
```
